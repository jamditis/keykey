Features

Built for people who make tutorials

Two display modes

Text stream shows stacked key labels that fade out. Visual keyboard highlights keys on a rendered layout. Switch between them from the tray menu or a hotkey.

Smart modifier handling

Filters incidental Shift taps during typing but catches intentional holds. Per-modifier overrides: always show, never show, combo only, or smart mode with a configurable threshold.

Multi-monitor

Follows the active window, follows the mouse cursor, or pins to a specific monitor and corner. Re-queries monitor geometry on every focus change. DPI-aware.

Full theme system

5 built-in presets. Every visual property is a CSS variable: fonts, colors, opacity, borders, shadows, animations. Changes apply instantly. Import and export themes as JSON.

Lightweight

Under 15MB installed, under 50MB memory. Uses the native WebView2 runtime instead of bundling Chromium. Less than 1% CPU when idle, under 3% during continuous typing.

System tray + hotkeys

Launches to tray, starts capturing immediately. All functions accessible from the tray menu. Optional global hotkeys with no hardcoded defaults, so nothing conflicts with your existing shortcuts.

Architecture

How it works

01

Capture

rdev installs a global keyboard hook via Win32 SetWindowsHookEx. Every key press and release is captured regardless of which window has focus.

02

Process

A Rust event processor applies smart mode logic: modifier hold timing, combo detection (Ctrl+Shift+S), repeat compression (A x3), and per-modifier filtering.

03

Emit

Processed events are sent to the frontend via Tauri's IPC event system. Two channels run in parallel: raw events for visual keyboard, processed events for text stream.

04

Render

The SvelteKit overlay renders keystrokes using the active display mode and theme. The overlay window is transparent, frameless, always-on-top, and click-through.

Install

Build from source

# Clone and build
git clone https://github.com/jamditis/keykey.git
cd keykey
npm install
npm run tauri build

# Or run in dev mode
npm run tauri dev

Requires Node.js 18+ and Rust (stable). Windows 10/11 only for now — macOS support is planned.
